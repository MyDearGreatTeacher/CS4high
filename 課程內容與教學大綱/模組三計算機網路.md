### 網路初入門
```


1.網際網路（Internet)
  1.1有線
   -有線電視 Internet 服務 
   -DSL (digital subscriber line)
   -Fiber to the Premises (FTTP)
  1.2無線
   -Wi-Fi (wireless fidelity)
   -行動寬頻
   -固定式無線 
   -衛星 Internet 服務
  1.3頻寬（bandwidth）
  1.4儲存單位
  1.5 IP 位址（IP address）
  1.6 網域名稱（domain name）
    1.6.1網域名稱系統（domain  name  system， DNS）
    1.6.2 TLD
    
2.全球資訊網（World Wide Web，WWW)
  2.1 網站（website）
  2.2 網站伺服器（web server）
  2.3 HTML（Hypertext Markup Language）
  2.4 瀏覽器（browser ）
  2.5 網址（web address)
  2.6 Web App 和 Mobile App
  2.7 網站的類型
  2.8 搜尋引擎（search  engine）
  2.9 社群網站（social networking site）
   -書籤網站（bookmarking site)
   -部落格（blog）
   -微博（microblog）
  2.10 串流化（streaming）
  2.11 入口網站（portal）
    -內容聚合（content  aggregation）網站
    
3.網路銀行和金融
    -網路銀行（online  banking）
    -線上交易（online trading）
  3.1 零售和拍賣
   -電子零售（e-retail，electronic retail ）
   -線上拍賣（online auction）
   -電子店面（electronic storefront）
  3.2 電子商務（e-commerce，electronic commerce)
  -企業對消費者（business-to-consumer，B2C）
  -消費者對消費者（consumer-to-consumer，C2C）
  -企業對企業（business-to-business，B2B）
 
4.多媒體（multimedia)
   -圖像
   -動畫
   -音訊
   -視訊
   -虛擬實境（VR）

5.網路通訊
  -電子郵件
  -電郵名單
  -網路傳訊
  -聊天室
  -線上討論區
  -VoIP
  -FTP
```
