### 系統軟體:作業系統 |公用程式
```
1.作業系統（operating system，OS）
  –工具或公用程式
  –系統軟體
  1.1使用者介面（user interface，UI）
    -圖形化使用者介面（graphical user interface，GUI）
    -命令列介面（command-line interface）
  1.2睡眠模式（sleep mode）
  1.3休眠模式（hibernate mode）
  1.4驅動程式（device driver）
  1.5隨插即用（Plug and Play）



2.記憶體管理（memory management）
   -虛擬記憶體（virtual memory）
   -效能監視器（performance monitor）是指負責評估並回報各種電腦資源與裝置相關資訊的程式。

3.作業系統分類
  3-1.單機作業系統（desktop operating system 或 stand-alone operating system）
     -Macintosh 作業系統
     -macOS
     -Chrome OS 
     -Windows
     -UNIX
     -Linux

  3-2.伺服器作業系統
     -Windows Server
     -macOS Server
     -UNIX
     -Linux

  3-3行動裝置作業系統
     -Android
     -iOS 
```
