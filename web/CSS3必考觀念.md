# CSS3十大必考標籤
### One
```
https://www.w3schools.com/css/tryit.asp?filename=trycss_syntax1
```
```
p {
  color: red;
  text-align: center;
}

所有<p>元素的字體將呈現為紅色並且置中對齊
```
```
<!DOCTYPE html>
<html>
<head>
<style>
p {
  color: red;
  text-align: center;
} 
</style>
</head>
<body>

<p>Hello World!</p>
<p>These paragraphs are styled with CSS.</p>

</body>
</html>
```

### Two
```
https://www.w3schools.com/css/tryit.asp?filename=trycss_syntax_id
```
```
#x1 {
  text-align: center;
  color: red;
}

id是x1的元素字體將呈現為紅色並且置中對齊
```
```
<!DOCTYPE html>
<html>
<head>
<style>
#x1 {
  text-align: center;
  color: red;
}
</style>
</head>
<body>

<p id="x1">Hello World!</p>
<p>This paragraph is not affected by the style.</p>

</body>
</html>
```
### Three
```
https://www.w3schools.com/css/tryit.asp?filename=trycss_background-color_elements
```
```
h1 {
  background-color: green;
}

div {
  background-color: lightblue;
}

p {
  background-color: yellow;
}

h1、div、p元素有三種不同的顏色
```
```
<!DOCTYPE html>
<html>
<head>
<style>
h1 {
  background-color: green;
}

div {
  background-color: lightblue;
}

p {
  background-color: yellow;
}
</style>
</head>
<body>

<h1>CSS background-color example!</h1>
<div>
This is a text inside a div element.
<p>This paragraph has its own background color.</p>
We are still in the div element.
</div>

</body>
</html>
```
